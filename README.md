# 📊 Sales Dashboard - Tableau

## 📌 Project Overview

This project analyzes sales data using Tableau. It includes KPI metrics, region-wise performance, category insights, and interactive filters.

## 🚀 Key Insights

* East region generates the highest sales
* South region has the lowest sales
* Technology is the top-performing category
